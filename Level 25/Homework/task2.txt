#ახსენით ტერმინები: დეკლარაცია, კუთვნილება მნიშვნელობა, CSS-ის წესი. მოიყვანეთ თითოეულზე მაგალითები

დეკლარაცია არის CSS-ის წესი, რომელიც განსაზღვრავს კონკრეტულ სტილურ მახასიათებელსა (property) და მის მნიშვნელობას (value). დეკლარაცია ყოველთვის იწერება

property: value;
color: red;
ეს დეკლარაცია ნიშნავს, რომ ტექსტის ფერი უნდა იყოს წითელი.

კუთვნილება (property) არის მახასიათებლის სახელი, რომელსაც CSS-ში ვიყენებთ ელემენტის სტილის შესაცვლელად.

 Proper         აღწერა

color	    ტექსტის ფერი
font-size	ფონტის ზომა
background	ფონის ფერი ან გამოსახულება
margin	    გარე დაშორება

font-size: 16px;(მაგალითი)

მნიშვნელობა არის კონკრეტული პარამეტრი, რომელიც გადაეცემა კუთვნილებას, რათა განსაზღვროს, როგორ იმოქმედოს.

  მნიშვნელობა	     კონტექსტი

red	ფერი         (color: red)
20px	          ზომა (font-size: 20px)
center	          მდებარეობა (text-align: center)

background-color: blue;(მაგალითი)

CSS-ის წესი შედგება სელექტორის, ფიგურული ფრჩხილების {} შიგნით მოთავსებული ერთი ან მეტი დეკლარაციისგან.

სტრუქტურა

selector {
    property: value;
    property: value;
}

მაგალითი

p {
    color: green;
    font-size: 18px;
}

შეჯამება:
  ტერმინი 	         აღწერა  	                             მაგალითი
დეკლარაცია	  კუთვნილება + მნიშვნელობა	                   color: red;
კუთვნილება	  სტილიზაციის მახასიათებლის სახელი	          color
მნიშვნელობა	   კუთვნილების მნიშვნელობა	                   red
CSS-ის წესი	    selector + დეკლარაციები	                    p { color: red; }
